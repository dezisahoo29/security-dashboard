# 🔐 AI-Powered Security Dashboard

## 📌 Overview

This project detects cyber threats using Machine Learning. It analyzes input features and predicts whether the activity is normal or malicious.

## 🚀 Features

* Anomaly Detection using Isolation Forest
* Threat Prediction API (Flask Backend)
* Interactive Dashboard UI (Frontend)

## 🛠️ Tech Stack

* Python
* Flask
* Scikit-learn
* HTML, CSS, JavaScript

## ▶️ How to Run

### Backend

cd backend
python app.py

### Frontend

Open frontend/index.html

## 📊 Output

* Prediction: 0 (Normal) / 1 (Attack)
* Threat Level: Low Risk / High Risk
